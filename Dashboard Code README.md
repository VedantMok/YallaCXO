# Yalla CXO

This Streamlit app visualizes a synthetic Dubai corporate opportunity dataset for fractional CXO services using Streamlit-native components and Altair charts instead of Plotly.

## Files to upload
- `app.py`
- `requirements.txt`
- `README.md`
- `fractional_cxo_dubai_top100_framework_ready.csv`
- `.streamlit/config.toml`

## Deploy on Streamlit Community Cloud
1. Create a GitHub repository.
2. Upload all required files.
3. Connect the repository in Streamlit Community Cloud.
4. Set the main file path to `app.py`.
5. Deploy.

## Dashboard sections
- Overview: KPI cards, role-demand charts, heatmaps, and priority account table.
- Segmentation: sector value and segmentation analysis.
- Pipeline: stage funnel, response analysis, and forecast table.
- Account Explorer: company-level drilldown and recommendation profile.
